# Projects

Navigator dei progetti personali e professionali di Alessio Sopranzi, pensato come complemento
di dettaglio al [CV](https://github.com/alesop95/my-cv) e alla
[tassonomia delle competenze](https://alesop95.github.io/skills/): il CV dichiara le competenze
e i progetti in forma sintetica, questo sito ne racconta il contesto tecnico per esteso, senza
dover leggere ogni singolo README su GitHub.

## Struttura

- **[Personal projects](personal/index.md)**: repository pubblici su GitHub, generati e
  aggiornati automaticamente da `scripts/update_personal_projects.py` a partire dai metadati
  reali di ciascun repository (descrizione, linguaggio principale, data dell'ultimo commit).
- **[Company projects](company/index.md)**: progetti professionali svolti per il datore di
  lavoro attuale, raccontati in forma **completamente anonimizzata** (nessun nome di azienda,
  cliente o sistema interno): scritti e rivisti a mano, mai generati automaticamente dal
  contenuto originale per motivi di riservatezza.
