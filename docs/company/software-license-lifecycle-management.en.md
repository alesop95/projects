# Software license migration and lifecycle management

**Sector**: language services and professional translation company

**Period**: to be confirmed

**Role**: IT Manager, systems administrator

**Technologies**: Windows Server, floating license management, software deployment to endpoints,
OCR, computer-assisted translation (CAT) tool

## Context

The company uses an optical character recognition (OCR) tool and a computer-assisted translation
(CAT) tool with floating licenses shared across multiple workstations: manual management of
versions, renewals and deployment to different endpoints risked misalignment and disruption for
end users.

## What was done

Full migration of the OCR tool to an updated version, and lifecycle management of the CAT tool's
floating licenses: migration phases, license versioning over time and distributed installation
across multiple endpoint workstations, to guarantee consistent availability of the tools to all
authorized users without exceeding concurrent license limits.

## Result

Service continuity for the users of both tools during and after the migration, with no
disruption for the teams that depend on them daily, and a repeatable license deployment process
for future versions.
