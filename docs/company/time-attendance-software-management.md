# Migrazione del software di rilevazione presenze

**Settore**: azienda di servizi linguistici e traduzione professionale

**Periodo**: da confermare

**Ruolo**: IT Manager, sistemista

**Tecnologie**: Windows Server, software di gestione presenze/timbrature

## Contesto

Il software di rilevazione delle presenze del personale necessitava di un intervento di
migrazione verso un'infrastruttura più solida, per restare affidabile nel tempo senza un referente
tecnico dedicato oltre l'IT Manager.

## Cosa è stato fatto

Migrazione del software di rilevazione presenze aziendale su un server Windows dedicato, con
relativa configurazione e passaggio in produzione per l'intero personale.

## Risultato

Rilevazione presenze affidabile per tutto il personale aziendale, senza interruzioni significative
di servizio durante e dopo la migrazione.
