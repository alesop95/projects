# Migración del software de control de presencia

**Sector**: empresa de servicios lingüísticos y traducción profesional

**Periodo**: por confirmar

**Rol**: IT Manager, administrador de sistemas

**Tecnologías**: Windows Server, software de gestión de presencia/fichajes

## Contexto

El software de control de presencia del personal necesitaba una migración hacia una
infraestructura más sólida, para seguir siendo fiable en el tiempo sin un referente técnico
dedicado más allá del IT Manager.

## Qué se hizo

Migración del software de control de presencia de la empresa a un servidor Windows dedicado, con
su correspondiente configuración y puesta en producción para todo el personal.

## Resultado

Control de presencia fiable para todo el personal de la empresa, sin interrupciones
significativas de servicio durante ni después de la migración.
