# Projects

Navegador de los proyectos personales y profesionales de Alessio Sopranzi, pensado como
complemento detallado del [CV](https://github.com/alesop95/my-cv) y de la
[taxonomía de competencias](https://alesop95.github.io/skills/): el CV declara las
competencias y los proyectos de forma sintética, este sitio cuenta su contexto técnico por
extenso, sin tener que leer cada README en GitHub.

## Estructura

- **[Personal projects](personal/index.md)**: repositorios públicos en GitHub, generados y
  actualizados automáticamente por `scripts/update_personal_projects.py` a partir de los
  metadatos reales de cada repositorio (descripción, lenguaje principal, fecha del último
  commit).
- **[Company projects](company/index.md)**: proyectos profesionales realizados para el
  empleador actual, contados de forma **totalmente anonimizada** (sin nombre de empresa,
  cliente o sistema interno): escritos y revisados a mano, nunca generados automáticamente a
  partir del contenido original, por motivos de confidencialidad.
