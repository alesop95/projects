# Projects

Navigator of Alessio Sopranzi's personal and professional projects, meant as a detailed
companion to the [CV](https://github.com/alesop95/my-cv) and to the
[skills taxonomy](https://alesop95.github.io/skills/): the CV states skills and projects
concisely, this site tells their technical context in full, without having to read every
single README on GitHub.

## Structure

- **[Personal projects](personal/index.md)**: public GitHub repositories, generated and
  updated automatically by `scripts/update_personal_projects.py` from each repository's real
  metadata (description, main language, last commit date).
- **[Company projects](company/index.md)**: professional projects carried out for the current
  employer, told in a **fully anonymized** form (no company, client or internal system name):
  written and reviewed by hand, never generated automatically from the original content, for
  confidentiality reasons.
