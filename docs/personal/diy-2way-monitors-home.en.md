This repository is a planning workspace for a DIY build of a pair of two-way studio monitor loudspeakers and matching active crossovers for home use, not yet an implemented project. As it stands, the local folder and the tracked repository contain only project scaffolding, written notes, reference links, and a working document on electroacoustics; there is no crossover design, no enclosure model, and no code committed so far, so this is best read as an early-stage personal research effort rather than a finished build.

The scope described in the project notes is an end-to-end electroacoustic design workflow: calibrated measurements of the actual listening position, driver and crossover design, room-mode analysis, and 3D acoustic simulation to predict on-axis and off-axis radiation behavior. The intended toolchain runs Akabak, a boundary-element acoustic simulator, under Wine on Ubuntu Studio, pairing an otherwise Windows-only measurement tool with a Linux audio workstation. Until measurement data and design files land in the repository, the project should be considered a documented intent rather than a working design.
