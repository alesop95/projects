# civitanext

_Sin descripción en GitHub._

- **Repositorio**: [alesop95/civitanext](https://github.com/alesop95/civitanext)
- **Lenguaje principal**: HTML
- **Última actualización**: 2026-07-06
- **Carpeta local**: `civitanext`

CivitaNext es un prototipo de diseño de alta fidelidad para una plataforma web y móvil de participación cívica dirigida a los jóvenes residentes de Civitanova Marche, que agrupa un calendario de eventos, cuestionarios cívicos, un foro de discusión, propuestas con votación, un perfil de miembro con carné digital, un panel de administración, y secciones de comunidad y ciudad como mentoría, un mapa de espacios cívicos y un archivo documental.

Lo que existe en este repositorio es explícitamente una entrega de diseño, no una aplicación en producción: los archivos JSX y HTML son maquetas con estilo React pensadas para reconstruirse dentro de una base de código real, y la propia documentación del proyecto declara que la autenticación está simulada con usuarios hardcodeados y sesiones en localStorage, y que todo el contenido, desde eventos hasta propuestas e hilos del foro, vive en arrays estáticos en memoria en lugar de en una base de datos. El lenguaje visual es deliberadamente distintivo: fondos color papel crema, sombras duras y desplazadas en lugar de difuminadas, gráficos dibujados a mano como estallidos y garabatos, y una combinación serif/sans (Source Serif 4 y Archivo) con un acento coral. El stack de destino recomendado es Next.js y React, pero, a fecha de este snapshot, no se ha construido ningún frontend funcional conectado a datos reales o a un backend.
