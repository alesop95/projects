CivitaNext is a hi-fidelity design prototype for a civic-participation web and mobile platform aimed at young residents of Civitanova Marche, bundling an event calendar, civic quizzes, a discussion forum, proposals with voting, a member profile with digital card, an admin panel, and community and city sections such as mentorship, a civic-spaces map, and a document archive.

What exists in this repository is explicitly a design handoff, not a production application: the JSX and HTML files are React-flavoured mockups meant to be rebuilt inside a real codebase, and the project's own documentation states that authentication is faked with hardcoded users and localStorage sessions, and that all content, from events to proposals to forum threads, lives in static in-memory arrays rather than a database. The visual language is deliberately distinctive: cream paper backgrounds, hard offset shadows instead of blurred ones, hand-drawn starburst and squiggle graphics, and a serif and sans pairing (Source Serif 4 and Archivo) with a coral accent. The recommended target stack is Next.js and React, but as of this snapshot no working frontend has been built against real data or a backend.
