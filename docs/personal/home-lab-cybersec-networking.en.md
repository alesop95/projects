This repository is currently a documentation and research collection for a home-lab networking and cybersecurity setup, not a coded project: the project's own instructions state explicitly that "the project has not yet been started as code" and that only the standard folder scaffolding is in place. What it does contain is reference material gathered while planning the lab: an OPNsense firewall feature whitepaper and a related quick-reference document, a draft network diagram (linked to an external draw.io file plus handwritten notes), a comparison list of privacy-respecting alternatives to mainstream cloud services (email, storage, maps, messaging), a link to a third-party write-up of a small enterprise-style network as a design reference, and a lengthy Word document sketching the home network and networking project itself.

In short, this is pre-implementation groundwork rather than a running system: firewall rules, VLAN segmentation, monitoring stack or any other code/config for the lab have not been committed yet, and the diagram is still a draft to be finalized in draw.io. It reads as a personal knowledge base an engineer builds before wiring up a home network with a dedicated firewall appliance and basic security monitoring, currently at the planning and reference-gathering stage.
