# totocalcio

A tool for investigating statistic pattern in the historical italian betting game

- **Repository**: [alesop95/totocalcio](https://github.com/alesop95/totocalcio)
- **Main language**: PowerShell
- **Last updated**: 2026-07-06
- **Local folder**: `totocalcio`

A very early-stage, exploratory project with no code written yet: the repository currently holds only planning documents distilling personal notes on Totocalcio, the Italian football-pool betting game, into a structured brief. The goal as scoped so far is to build a dataset from official betting bulletins (available weekly since January 2022), run statistical analysis and visualization over it, and eventually derive a custom prediction algorithm that conditions the set of playable combinations, mirroring concepts from existing Totocalcio software such as AND/OR/link conditioning and integral-column reduction, but making the underlying statistics and trade-offs visible instead of opaque.

The documentation is explicit that this is a preliminary phase: open questions include how to structure the bulletin dataset, unclear Excel-based methods referenced from external sources, and whether the classic 14-match format is still playable at all, something the author notes still needs checking in person at a betting shop. There is no dataset schema, no script, and no prototype yet, so this project is best described as a well-organized research brief rather than working software.
