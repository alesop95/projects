This browser-based web app layers automatically generated harmonics on top of notes played from a MIDI keyboard, choosing which harmonics to add based on hand movement captured by a BBC micro:bit worn as a glove and connected over Bluetooth Low Energy. Roll and pitch rotations of the hand select the harmonic interval and its octave, a 3D hand model rendered with Three.js mirrors the movement live, and the audio itself is synthesized entirely client-side rather than relying on prerecorded samples, so the app also works without the glove attached, driven from the keyboard alone.

The interface is built with Vue.js for the reactive controls (ADSR envelope sliders drawn as SVG, timbre selection, a spectrum display of the generated signal), while the actual sound engine handles MIDI note-on and note-off events and drives one oscillator per active harmonic through the Web Audio API. A separate MATLAB script was used offline to analyze real recorded instrument samples and decide, per timbre, how many harmonics can be reproduced convincingly in a web context; the resulting harmonic counts are hard-coded into the JavaScript synth. The README documents this as a project built to explore gesture-controlled sound synthesis in the browser, including honestly listed follow-up work, such as live pitch shifting, that was never implemented.
