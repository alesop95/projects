X CROSS Training is, at this stage, a specification and dataset package for a Flutter mobile app for programming cross-training/functional workouts, not yet an implemented application: the repository explicitly states there is no application code, only documents and JSON data extracted from a source PowerPoint deck and an Excel workbook.

The most concrete technical artifact is the design for the core combination-generation algorithm: workouts are built by combining exercises from three families (weightlifting, gymnastics, locomotion) into non-repeating one-, two-, or three-element combinations across a training cycle. The design is derived directly from a working Excel prototype that used RANDBETWEEN over a shrinking range to draw combinations without replacement, documented here as a Fisher-Yates variant and translated into a Dart class specification (GeneratoreCombinazioni) with filtering by available equipment, athlete level, and enabled exercise groups. The rest of the repository lays out data models, workout modes (EMOM, Tabata, AMRAP, For Time), and an MVP roadmap. It reads as thorough upfront planning handed off for development rather than a working product.
